# チキングラタン

鶏もも肉とブロッコリーをホワイトソースで仕上げ、チーズをのせて焼く定番グラタン。

- 鶏もも肉 … 150g
- ブロッコリー … 約200g
- 牛乳 … 1と1/2カップ
- にんにく（すりおろし）… 1/2片
- ピザ用チーズ … 適量
- バター … 10g
- 小麦粉 … 大さじ2
- 塩 … 小さじ2/3

1. ブロッコリーは小さめに分けて下茹でする。鶏もも肉は2cm角に切って塩でもむ
2. フライパンにバターを中火で熱し、鶏肉とブロッコリーを炒める
3. 小麦粉をふり入れて軽く炒め、牛乳を少しずつ加えながら混ぜる。とろみがついたら塩・にんにくで味を整える
4. 耐熱皿に入れてチーズをのせ、オーブンレンジのマカロニグラタンモードで焼く

参考：長谷川あかり
