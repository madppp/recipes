# ラー油

ホールスパイスと香味野菜を油で煮出して作る自家製ラー油。

## 材料

### A（香味野菜）
- 長ねぎ（青い部分） … 1本
- 生姜 … 1と1/3かけ（20g）
- 赤唐辛子ホール … 6本

### B
- サラダ油 … 300g

### C（ホールスパイス）
- 花椒ホール … 5g
- 八角 … 2g
- シナモン … 1g
- クローブ … 1g
- カルダモン … 1g
- フェンネルシード … 少々

### その他
- 一味唐辛子 … 30g

1. A・B・Cをすべて鍋に入れ、ふつふつと泡が出る程度の火で **30分** 加熱する
2. ホールスパイス（C）と赤唐辛子（A）をとりのぞく
3. 中火で加熱し **170℃** まで上げて火をとめる
4. ボウルに一味唐辛子を入れ、③を注いでよくかき混ぜる
5. さめたらビンに移す

**ポイント:** 一味唐辛子はあとでとったほうがいいかも。
